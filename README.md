# Wumpus

Una simple prueba para completar el proceso de selección para MYSPHERA.

Simplemente haz un 'fork' del repositorio en tu cuenta e implementa el juego del Wumpus,
tal y como se describe en el pdf incluido en el repositorio.

Utiliza para el desarrollo Eclipse y Java, ambos con las versiones con la que te sientas
más cómodo.
